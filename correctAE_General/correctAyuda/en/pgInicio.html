CorrectA Help

Application for the evaluation of multiple-choice, multi-template exams.

Application main window. Here is the table containing all the answers sheets read. It has the following fields:

  • Num.

    • Reading order number.

  • Personal I.D.

    • Student id.

  • Type

    • Test type, which corresponds to the answers template that will be applied this test. There can be up to six of these types. Types equivalences table specifies its correspondence with the main type (type 0).
    Important: If the type value has not been read correctly, that answer sheet will not be evaluated.

  • Group

    • Group to which the student belongs (optional).

  • Answers

    • String of student answers to all questions. It contains the answer option letter marked, one for each question. Blank means the student did not answered this question. A symbol "?" appears when there is more than one mark on that answer.

  • File

    • Path to the graphics file containing this answer sheet.

Note: By double-clicking on a table row you can see the original graphic file and the interpretation of the marks that the application has made for this test. Several can be opened simultaneously.
