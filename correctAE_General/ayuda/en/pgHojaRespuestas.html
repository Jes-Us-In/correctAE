Answers Sheet

Answers Sheet Model.

The model has been designed with an arrangement of its elements to improve the interpretation of the answer sheets and the accuracy of the results.

The answer sheet model has five fields:

  1. Answers:

    There are 100 valid location response marks. The qunatity in use is defined in the table of types and equivalences.

  2. I.D. (number):

    Identification number, 8 digits.

  3. I.D. (letter):

    Identification letter.

  4. Test type:

    Type number, a value from 1 to 6 that identificate the type of test. It is essential, to correct the answer sheet. Indicate the equivalences of the answers in relation to the master type (type 0).

  5. Group:

    An optional field that can be used to identify/classify answer sheets. It's also a value from 1 to 6.

There are others fields such as firstname, lastname, etc., which are not interpreted but are necessary to the correct student identification, attending to subsequent reclamations and others situations.

Brief instructions are also included to fill out the answer sheet correctly and avoid reading/interpretation errors as much as possible.





On this application window you can see the test model, already described, and make two important actions via two additional buttons:

  • Change Logo:

    Change the answer sheet logo to one of our choice. It is important that the pixel size remains the same, otherwise it will be rescaled, to avoid deforming the model. The size must be 256 by 50 pixels.

  • Print Template:

    This buttom shows the system print dialog, to print the necessary amount of answer sheet.
